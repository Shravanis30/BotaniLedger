# Contributing to BotaniLedger

First off, thank you for considering contributing to BotaniLedger! It's people like you that make BotaniLedger such a great tool for the transparency in the botanical industry.

## 🌈 Code of Conduct

Help us keep BotaniLedger open and inclusive. Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md).

## 🚀 How Can I Contribute?

### Reporting Bugs
- Use the GitHub issue tracker.
- Describe the bug and include steps to reproduce it.
- Include details about your environment (OS, Node version, etc.).

### Suggesting Enhancements
- Check if the feature has already been suggested.
- Open an issue with the "enhancement" label.
- Provide a clear and concise description of the feature.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Install dependencies with `npm install`.
3. If you've added code that should be tested, add tests.
4. Ensure the test suite passes.
5. Format your code with Prettier.
6. Write a clear pull request description.

## 🛠️ Development Workflow

- **Backend**: `npm run backend`
- **Frontend**: `npm run dev`
- **Linting**: `npm run lint`

## 🎨 Style Guide

- Use functional components and hooks for React.
- Use Tailwind CSS for all styling.
- Follow the existing project structure.
- Write descriptive commit messages.
